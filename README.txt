Exercices C00 - Piscine Codeloccol

Ce dépôt contient des scripts Bash pour découvrir les bases du terminal sous Unix/Linux. Ces exercices ont été réalisés dans le cadre de la Piscine Codeloccol.

Objectifs pédagogiques :
- Utiliser les commandes de base du terminal
- Automatiser des tâches simples avec des scripts Bash
- Manipuler des fichiers et des répertoires

Liste des exercices :

1. hello_world.sh        -> Affiche "Hello, World!" à l'écran
2. current_datetime.sh   -> Affiche la date et l'heure actuelles
3. list_files.sh         -> Liste les fichiers du répertoire courant
4. create_directory.sh   -> Crée un dossier nommé test_directory
5. check_file_exist.sh   -> Vérifie si un fichier donné existe ou non
